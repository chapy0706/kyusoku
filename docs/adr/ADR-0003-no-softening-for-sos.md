# ADR-0003: SOSレベルの投稿はマイルド化しない

- 日付: 2026-05-27
- ステータス: 承認済み

---

## コンテキスト

kyusoku は送信テキストを「加工して」担当営業に届けると約束している。この加工（マイルド化）は、強い語彙や感情的な表現をやわらかく変換することで、読む側が冷静に受け取れるようにするための仕組みだ。

しかし、リスクレベル3（SOS）に分類された投稿にも同じ処理を施すと、深刻な問題が起きる。「もう限界です、消えたい」が「少し疲れているようです」に変換されてしまえば、担当営業は緊急性を感じ取れない。マイルド化という優しさが、最も届けるべき声の切実さを奪ってしまう。

## 決定

リスクレベル3（SOS）の投稿はマイルド化の対象外とする。

管理画面にはマイルド化前の表現に近い形で届け、「即時対応」のアラートと合わせて表示する。やわらかさと、伝わるべき切実さは使い分ける。

## 根拠

- マイルド化の目的は「日常の愚痴や強い言い回しを和らげること」であり、「危機のサインを隠すこと」ではない
- SOSレベルの緊急性は、受け取る担当営業がそのまま感じ取れる必要がある
- 完全な原文表示ではなく「切実さを保った形」とすることで、読む側の受け取りやすさも一定確保する

## 影響

- マイルド化処理にリスクレベルを入力として渡す必要がある（判定が先である理由とも対応。ADR-0002参照）
- 管理画面のSOS表示UIは、他のレベルと視覚的に明確に区別する
- 将来的にレベル2（警告）の扱いも再検討する余地がある。その際は新しいADRを立てる
